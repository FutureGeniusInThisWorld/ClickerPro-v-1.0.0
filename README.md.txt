# Auto Clicker Pro

Простой автокликер на Python с GUI на основе tkinter.

Возможности
- Автоматические клики мышью
- Настраиваемый интервал между кликами
- Горячие клавиши (F6, F7, F8)
- Чёрный интерфейс

Установка

```bash
pip install -r requirements.txt

запуск:
python clickerpro.py

f6 - start
f7 - stop
f8 - switch

can't work in theme mode

exe сборка
pip install pyinstaller
pyinstaller clickerpro.spec
